# プロジェクト仕様書：bunchoniki Store (AltStore Japan Wrapper)

## 1. サイトコンセプト
- **名称:** bunchoniki Store（文鳥ストア）
- **役割:** 英語圏の AltStore エコシステムを日本人が安心して利用するための日本語ラッパーポータル。
- **スローガン:** 「iPhoneに、自由と安心を。法律が認めた新しいアプリ体験。」
- **ブランディング:** 文鳥をモチーフにした、清潔感と信頼感のある白・ライトブルー基調のデザイン。

## 2. 記事構成（アドセンス審査・SEO対策）
以下の3層構造（計10記事）を初期コンテンツとして生成する。

### 第1層：法律と安全性（信頼の土台）
1. スマホ新法解説：iPhoneのアプリ配布が変わる理由
2. Apple公証（Notarization）とは：サイドローディングが安全な仕組み
3. 運営方針：当サイトが「公証済み」にこだわる理由（著作権遵守宣言）

### 第2層：具体的な利益（ユーザー訴求）
4. 手数料30%カットの行方：アプリ価格は安くなるのか？
5. 決済の自由化：App Store決済とWeb決済の賢い使い分け
6. 直接支援のメリット：お気に入りアプリの継続を支えるパトロン文化

### 第3層：アプリ紹介（SEO目玉）
7. Delta/UTM SE：iPhoneでレトロゲームやPCを動かす公式の方法
8. Sidestore：PC不要でアプリ管理する日本語ガイド
9. 【2026最新】iPhoneの自由を広げる公証済み神アプリ3選
10. サイドローディングFAQ：不安とエラーへの回答集

## 3. 機能要件
- **動的カタログ:** `apps.json` をフェッチし、各アプリの紹介カードを表示。
- **公証済みバッジ:** 各アプリページに独自デザインの安全確認バッジを表示。
- **インポート誘導:** 日本語の確認モーダルを経て `altstore://` スキームへ誘導。
- **SEO最適化:** `SoftwareApplication` 構造化データの埋め込み。
- **免責事項:** エミュレータ等の利用に関する著作権遵守の注釈を各所に配置。

## 4. 技術スタック & ルール
- **技術:** GitHub Pages, Tailwind CSS, Vanilla JS.
- **ロゴ:** 白い文鳥のSVGアイコン（インラインで使用）。
- **禁止事項:** コードへのコメント記入は一切禁止。
- **広告:** Google AdSense 用のプレースホルダーをヘッダー・記事間・フッターに設置。
