install_desktop AltStoreでアプリをインストールする方法

PC/Mac準備から初回セットアップ、実際のアプリインストールまで完全ガイド

lightbulb AltStoreでアプリを入れるって難しい？

「AltStoreの話をよく聞くけど、実際にアプリを入れるには何をすればいいの？」 「PC/Macって絶対に必要なの？」 「インストール後のメンテナンスはどうするの？」

こんな疑問をお持ちの方は多いでしょう。実は、AltStore でアプリをインストールするのは、 App Store よりもシンプルで直感的です。この記事では、 初心者の方でも迷わないようにステップバイステップで解説します。

記事を読み終わる頃には、最初のアプリをインストールし、 定期メンテナンスの方法も理解できているはずです。

checklist 事前準備：インストール前に確認すること

iOS のバージョン要件

AltStore はiOS 14.4 以上で動作します。 お手持ちの iPhone が iOS 14.4 以上かどうか、設定アプリから確認してください。

  • 設定 → 一般 → ソフトウェア・アップデート
  • 現在のバージョンが表示されます

PC / Mac の用意

AltStore をインストールするには、Windows PC または Macが必要です。 初回セットアップ時のみ USB 接続が必要で、その後は Wi-Fi 経由での操作が可能になります。

  • Windows: Windows 10 以上推奨
  • Mac: macOS 10.14 以上推奨

Apple ID の用意

有効な Apple ID（2段階認証を有効にしたもの）が必要です。 既に iPhone で使用している Apple ID で問題ありません。

info 重要：Developer Accountは不要

有料の Developer Account は必要ありません。無料の Apple ID で大丈夫です。

USB ケーブル（初回セットアップのみ）

cable 初回のみUSB接続が必要

初回セットアップ時のみ、iPhone をPC/Mac に USB で接続します。 その後は Wi-Fi 経由での操作になるため、毎回 USB 接続は不要です。

download ステップ 1：AltServer をダウンロード＆インストール

AltServer とは

AltServerは、PC/Mac で動作するツールで、 AltStore にアプリをインストール・更新するために必要なアプリケーションです。 いわば「App Store の配信役」のような存在だと考えてください。

ダウンロード手順

  1. AltServer 公式サイトを開く
  2. お手持ちの OS（Windows / Mac）に合わせてダウンロードボタンをクリック
  3. ダウンロードしたインストーラーを実行

Windows での インストール例

ダウンロードした .exe ファイルをダブルクリック → 指示に従ってインストール完了。 特に特殊な設定は必要ありません。

Mac でのインストール例

ダウンロードした .dmg ファイルをダブルクリック → AltServer アイコンを Applications フォルダにドラッグ&ドロップ。 その後、Applications フォルダから AltServer を実行してください。

login ステップ 2：AltServer に Apple ID でログイン

AltServer を起動

インストール完了後、AltServer を起動します。 メニューバー（Windows はシステムトレイ）に AltServer のアイコンが表示されます。

Apple ID でサインイン

AltServer のメニューを開き、「Sign In」または「ログイン」を選択します。 iPhone で使用している Apple ID（メールアドレスとパスワード）を入力してください。

2段階認証への対応

Apple ID に2段階認証が有効な場合、 確認コードの入力が求められます。 お手持ちの別の Apple デバイス（iPad など）に確認コードが送られるため、それを入力してください。

cable ステップ 3：iPhone をPC/Mac に接続して初回セットアップ

USB ケーブルで接続

iPhone を USB ケーブルで PC/Mac に接続します。

check_circle 「信頼」をタップすることを忘れずに

「このコンピューターを信頼しますか？」というメッセージが iPhone に表示される場合は、 必ず「信頼」をタップしてください。これをしないとインストールが進みません。

AltServer から「Install AltStore」を選択

AltServer のメニューから「Install AltStore」を選択します。 すると、PC/Mac が iPhone に AltStore アプリをインストールし始めます。

インストール完了を待つ

数秒～数十秒でインストールが完了します。 iPhone のホーム画面に「AltStore」というアプリが追加されていることを確認してください。

Apple ID で iPhone 上の AltStore にサインイン

iPhone 上の AltStore アプリを起動し、 先ほど AltServer に入力した Apple ID でサインインします。 これで初回セットアップは完了です。

apps ステップ 4：実際にアプリをインストール

AltStore 内でアプリを探す

iPhone 上の AltStore アプリを開くと、 複数のアプリが表示されます（Tweaked Apps、Emulators など）。 インストールしたいアプリを探してタップします。

アプリの詳細ページから「Get」をタップ

アプリの詳細ページに進むと「Get」ボタンが表示されます。 このボタンをタップするとダウンロードが始まります。

ダウンロード完了を待つ

ダウンロードが完了すると「Open」ボタンに変わります。 タップするとアプリが起動します。 数秒待つと、App Store のアイコンのように iPhone のホーム画面に追加されます。

別の方法：IPA ファイルから直接インストール

AltStore 内のアプリ一覧以外にも、IPA ファイル（iPhone アプリのパッケージファイル） を PC/Mac で用意して、AltServer からインストールする方法もあります。 これは、カスタムアプリや個人開発アプリをインストールしたい場合に便利です。

詳しくは 「サイドローディングとは」 をご覧ください。

wifi ステップ 5：Wi-Fi 経由での自動更新設定（推奨）

Wi-Fi 同期を有効化する理由

autorenew 自動更新で手間いらず

AltStore のアプリは7日ごとに署名を更新する必要があります。 毎回 USB で接続する代わりに、Wi-Fi 経由で自動更新するように設定すると、 手動操作がほぼ不要になります。

AltServer 側での設定

PC/Mac の AltServer メニューから、インストール済みの iPhone を選択します。 「Sync with [iPhone name] over Wi-Fi」オプションをオンにしてください。

前提条件

warning Wi-Fi同期を機能させるための必須条件
  • PC/Mac と iPhone が同じ Wi-Fi ネットワークに接続していること
  • AltServer がバックグラウンド起動していること
  • iPhone が同じ Wi-Fi に常に接続していることが理想的

自動更新の仕組み

Wi-Fi 同期を有効にすると、AltServer は以下の処理を自動的に行います：

  1. 毎日、iPhone 上のアプリの署名有効期限をチェック
  2. 期限切れ間近のアプリを検出
  3. Wi-Fi 経由で自動的に再署名（更新）
  4. ユーザーは何もしなくてOK

つまり、PC を起動して AltServer を実行しておくだけで、 アプリは常に最新状態に保たれるわけです。

build トラブル対応：よくある問題と解決策

Q. インストールが失敗する

troubleshoot 基本的なチェック項目

以下を確認してください：

  • Apple ID のパスワードが正しく入力されているか
  • iOS が 14.4 以上か
  • iPhone の容量は十分か（アプリサイズ以上の空きが必要）
  • AltServer を再起動してみる

Q. インストール後、アプリが起動しない

error 署名が切れている可能性があります

解決法：AltStore アプリを開き、該当アプリの横に表示されている更新ボタン（↻）をタップして、 署名を更新してください。通常、数分で起動可能になります。

Q. Wi-Fi 自動更新がうまくいかない

wifi_off Wi-Fi同期のチェックポイント

以下を確認してください：

  • PC/Mac と iPhone が同じ Wi-Fi ネットワークに接続しているか
  • AltServer が PC 上でバックグラウンド実行中か（メニューバーに表示されているか）
  • Wi-Fi ルーターのファイアウォール設定が AltServer の通信をブロックしていないか
  • PC を再起動して、再度 Wi-Fi 同期をオンにしてみる

Q. PC がない場合はどうするの？

A. 家族や友人の PC/Mac を借りて初回セットアップを完了させれば、 その後は iPhone 上で手動で署名を更新できます。 毎回 USB 接続は不要です。

Q. Apple ID が複数ある場合は？

A. AltServer と AltStore に同じ Apple ID でサインインしてください。 複数の Apple ID は対応していません。

update 定期メンテナンス:署名更新の方法

手動更新（Wi-Fi 同期なし）

AltStore アプリを開き、署名を更新したいアプリの横に表示されている更新ボタン（↻）をタップします。 数分で完了します。

Wi-Fi 経由の自動更新（推奨）

前述の「ステップ 5」の設定を完了していれば、 何もしなくても自動的に更新されます。 PC を起動して AltServer を実行しているだけで大丈夫です。

更新の頻度は？

schedule 重要：7日ごとの署名更新を忘れずに

AltStore のアプリは7日ごとに署名が切れます。 Wi-Fi 同期が有効なら、期限切れの2～3日前に自動更新されます。

手動更新の場合は、7日以内なら随時更新できます。期限を過ぎるとアプリが起動しなくなりますので注意してください。

done_all まとめ

  • 事前準備： iOS 14.4以上、PC/Mac、有効な Apple ID を用意する必要があります。
  • セットアップ： AltServer をインストール → Apple ID でサインイン → 初回は USB 接続で AltStore をインストール。 その後は Wi-Fi で操作可能です。
  • アプリのインストール： iPhone 上の AltStore アプリから好きなアプリを選んで「Get」をタップするだけ。 App Store と同じくらい簡単です。
  • メンテナンス： Wi-Fi 自動更新を設定すれば、ほぼ手放しで署名管理ができます。 PC を常に起動する必要はありません。
  • トラブル時： ほとんどの問題は Apple ID の確認や PC の再起動で解決します。
