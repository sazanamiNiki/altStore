sports_esports Delta/UTM SE 完全ガイド

iPhoneでレトロゲームやWindows/Linuxを動かす公式の方法

lightbulb はじめに

2024年4月、AppleはApp Storeでのエミュレータ配布ポリシーを緩和し、ゲームエミュレータや仮想化アプリの配布が可能になりました。これにより、Delta EmulatorやUTM SEといったアプリが公式に入手できるようになり、iPhoneやiPadでレトロゲームを楽しんだり、Windows/Linuxを実行したりすることが、脱獄なしで可能になりました。

本ガイドでは、Riley Testut氏が開発したDelta Emulatorと、QEMUベースの仮想化アプリUTM SEの詳細な使い方を解説します。どちらもAppleの公証を受けた安全なアプリであり、App StoreまたはAltStoreから入手可能です。

videogame_asset Part 1: Delta Emulator

1.1 Delta Emulatorとは

Delta Emulatorは、iOS向けに開発されたオールインワンのレトロゲームエミュレータです。かつて人気を博したGBA4iOSの後継として開発され、現在ではApp Storeから正式にダウンロード可能です。開発者のRiley Testut氏は、AltStoreの開発者としても知られています。

Deltaは複数のオープンソースエミュレータコアを統合しており、以下のゲーム機に対応しています：

  • Nintendo Entertainment System (NES) - Nestopiaコアを使用
  • Super Nintendo (SNES) - Snes9xコアを使用
  • Nintendo 64 (N64) - mupen64plusコアを使用（OpenGL ES 3.0対応）
  • Game Boy / Game Boy Color - Gambatteコアを使用
  • Game Boy Advance (GBA) - VisualBoyAdvance-Mコアを使用
  • Nintendo DS - melonDSコアを使用
  • Sega Genesis - Genesis Plus GXコアを使用（有料版のみ）

1.2 インストール方法

Delta EmulatorはiOS 12以降を搭載したiPhone/iPadで動作します。macOSの場合はmacOS 11.0以降かつApple M1チップ搭載のMacが必要です。

方法1: App Storeからダウンロード（米国アカウント）

米国のApp Storeアカウントをお持ちの場合、「Delta - Game Emulator」で検索してダウンロードできます。日本のApp Storeでは現時点で公開されていないため、地域設定の変更または代替方法が必要です。

方法2: AltStore経由でインストール

日本在住のユーザーには、AltStore経由でのインストールが最も確実な方法です。以下の手順で進めてください：

  1. PCにAltServerをインストール（Windows 10以降、またはmacOS）
  2. iPhoneをPCに接続し、AltStoreをインストール
  3. AltStore内の「Browse」タブでDelta Emulatorを検索
  4. 「GET」をタップしてインストール完了

方法3: AltStore PAL（欧州ユーザー向け）

EUに居住している場合、AltStore PALを利用することでPCなしでDeltaをインストールできます。EUのデジタル市場法（DMA）により、サードパーティのアプリストアがiOSでも認められています。

1.3 ゲームROMの追加方法

Delta自体にはゲームROMは含まれていません。法的に所有しているゲームのROMファイルを追加する必要があります。対応フォーマットは各プラットフォームの標準的な形式（.nes, .sfc, .z64, .gb, .gbc, .gba, .nds, .mdなど）です。

  1. ROMファイルをiPhone/iPadの「ファイル」アプリに保存
  2. Deltaを起動し、右上の「+」ボタンをタップ
  3. 「ファイル」から該当するROMファイルを選択
  4. 自動的にライブラリに追加されます

Nintendo DSについて：以前はBIOSファイルが必須でしたが、現在のバージョンではBIOSファイルはオプションとなっています。また、GBAカートリッジを「挿入」するデュアルスロット機能にも対応しています。

1.4 コントローラー設定とマルチプレイヤー

Deltaは多様なコントローラーに対応しています：

  • PlayStation 4 (DualShock 4)
  • PlayStation 5 (DualSense)
  • Xbox One S / Xbox Series X|S
  • MFi認証コントローラー
  • 画面上のバーチャルコントローラー（カスタムスキン対応）

ローカルマルチプレイヤー：NES、SNES、N64のゲームでは最大4人までのローカルマルチプレイヤーに対応。各プレイヤーが別々のコントローラーを接続することで、友人や家族と一緒にプレイできます。

オンラインマルチプレイヤー：Nintendo DSのオンライン対応ゲームでは、Wiimmfi、WiiLink WFC、AltWFCなどのコミュニティサーバーを経由して世界中の友人とオンラインプレイが可能です。

1.5 セーブ機能とクラウド同期

Deltaには強力なセーブ機能が搭載されています：

  • 通常セーブ：ゲーム内のセーブ機能を通常通り使用可能
  • ステートセーブ：任意のタイミングでゲーム状態を保存・復元
  • オートセーブ：バックグラウンドに移行時に自動的に状態を保存
  • クラウド同期：iCloud、Google Drive、Dropboxに対応し、複数デバイス間でセーブデータを共有

1.6 Delta 2.0の新機能（2025-2026）

最新のDelta 2.0ベータ版では、以下の新機能が追加されています：

  • Liquid Glass UI：iOS 26のデザイン言語に対応したモダンなインターフェース
  • RetroAchievements：GBC、GBA、DSゲームで実績機能が利用可能
  • Sega Genesis対応：新しいコントローラースキンとともにメガドライブゲームをサポート
  • Game Boy Camera機能：iPhoneのカメラを使用してレトロなGame Boy Camera体験を再現

computer Part 2: UTM SE

2.1 UTM SEとは

UTM SEは、QEMUベースのフル機能システムエミュレータおよび仮想マシンホストです。iPhoneやiPadでWindows、Linux、その他のOSを実行することができます。「SE」は「Slow Edition」の略で、App Storeの制限に対応するためJITコンパイルの代わりにTCGインタープリターを使用しています。

2024年に登場したUTM SEは、iOS App Store初のPCエミュレータとなりました。通常のUTMはサイドローディングで入手可能ですが、AppleのJIT制限によりApp Storeでは配布できませんでした。UTM SEはこの制限を回避するため、パフォーマンスは低下しますが、脱獄やJIT回避策なしで使用可能です。

2.2 対応アーキテクチャ

UTM SEはQEMUが対応する30以上のプロセッサアーキテクチャをエミュレート可能です：

  • x86_64 / i386：Windows、Linuxの多くのディストリビューション
  • ARM64 / ARM：ARMベースのOS、Raspberry Pi OS
  • RISC-V：RISC-V64 Ubuntuなど
  • PowerPC：クラシックMac OS 8.x/9.x
  • その他：MIPS、SPARC、m68kなど

2.3 インストールとセットアップ

UTM SEはiOS 14.0以降を搭載したデバイスで動作します。App Storeから「UTM SE」で検索してダウンロードするか、AltStore経由でインストールできます。

仮想マシンの作成手順

  1. UTM SEを起動し、「+」ボタンをタップ
  2. 「Emulate」を選択（iOSでは仮想化は利用不可）
  3. OSの種類を選択（Windows、Linux、Other）
  4. ISOファイルを選択または後で追加を選択
  5. RAM、CPUコア数、ストレージサイズを設定
  6. 作成完了後、仮想マシンを起動

2.4 Windows のインストール

UTM SEでは様々なバージョンのWindowsを実行できますが、パフォーマンスの制約があるため、軽量なOSが推奨されます。

Windows 98/XPの場合（推奨）

Windows 98はUTM SEで驚くほどスムーズに動作します。M1 iPad Proでは、Pentium 120相当のパフォーマンスが得られます。90年代のWindowsソフトウェアやDOSゲームを楽しむには最適な選択です。

  • 推奨RAM: 512MB - 1GB
  • 推奨ストレージ: 2-4GB
  • 「Force PS/2 Controller」を有効にすると起動時のハングを防止

Windows 10/11の場合

技術的には動作しますが、JITなしのエミュレーションではパフォーマンスが大幅に低下します。基本的な操作は可能ですが、実用的とは言いがたい速度です。

  • 推奨RAM: 4-8GB
  • 推奨ストレージ: 60-100GB
  • GPU仮想化非対応のためDirectX/OpenGLは使用不可

2.5 Linuxのインストール

軽量Linuxディストリビューションは、UTM SEで比較的良好に動作します。以下のディストリビューションがおすすめです：

  • Alpine Linux：非常に軽量で高速起動
  • Tiny Core Linux：最小限のリソースで動作
  • Debian（最小インストール）：安定性重視
  • Ubuntu Server：GUIなしで使用推奨

2.6 パフォーマンス最適化のヒント

  • GUIを無効化：コマンドラインのみで使用するとパフォーマンス向上
  • VirtIO使用：ARM仮想マシンではVirtIOドライバでI/O性能向上
  • QCOW2形式：ディスクイメージはQCOW2形式を使用
  • 不要なサービス停止：バックグラウンドプロセスを最小限に
  • スワップ無効化：仮想マシン内のスワップは無効に

2.7 UTM SEの最新アップデート（2025-2026）

最新バージョンでは以下の改善が行われています：

  • QEMU v10.0.2：バグ修正とパフォーマンス改善
  • Liquid Glassデザイン：iOS 26のUI言語に対応
  • Shortcuts対応：自動化ワークフローとの統合
  • カスタムキーボードショートカット：Ctrl+Alt+Delなどを簡単に送信
  • 改善されたウィザード：RISC-V64 Ubuntu、クラシックMac OS 8.x/9.x、Windows 95/98の簡単セットアップ

done_all まとめ

Delta EmulatorとUTM SEは、iPhoneやiPadの可能性を大きく広げるアプリです。Deltaを使えば、NES、SNES、N64、Game Boy、Nintendo DSなど、懐かしいゲームを高品質で楽しむことができます。UTM SEでは、Windows 98などのレトロOSや軽量Linuxを実行し、モバイルデバイスで様々なソフトウェアを試すことが可能です。

どちらのアプリもAppleの公証を受けており、安全に使用できます。2026年は「iOSでのレトロゲーミング史上最高の年」と言われており、これらのアプリを活用して、iPhoneでの体験を最大限に楽しんでください。
