sports_esports Delta/UTM SE 完全ガイド

iPhoneでレトロゲームやWindows/Linuxを動かす公式の方法

lightbulb はじめに

2024年4月、AppleはApp Storeでのエミュレータ配布ポリシーを緩和し、ゲームエミュレータや仮想化アプリの配布が可能になりました。これにより、Delta EmulatorやUTM SEといったアプリが公式に入手できるようになり、iPhoneやiPadでレトロゲームを楽しんだり、Windows/Linuxを実行したりすることが、脱獄なしで可能になりました。

本ガイドでは、Riley Testut氏が開発したDelta Emulatorと、QEMUベースの仮想化アプリUTM SEの詳細な使い方を解説します。どちらもオープンソースで透明性が確保された安全なアプリであり、App StoreまたはAltStoreから入手可能です。

videogame_asset Part 1: Delta Emulator

Delta Emulatorは、iOS向けに開発されたオールインワンのレトロゲームエミュレータです。かつて人気を博したGBA4iOSの後継として開発され、現在ではApp Storeから正式にダウンロード可能です。開発者のRiley Testut氏は、AltStoreの開発者としても知られています。

Delta EmulatorはiOS 12以降を搭載したiPhone/iPadで動作します。macOSの場合はmacOS 11.0以降かつApple M1チップ搭載のMacが必要です。

storefront 方法1: App Storeからダウンロード（米国アカウント）

米国のApp Storeアカウントをお持ちの場合、「Delta - Game Emulator」で検索してダウンロードできます。日本のApp Storeでは現時点で公開されていないため、地域設定の変更または代替方法が必要です。

download 方法2: AltStore経由でインストール（日本ユーザー推奨）

日本在住のユーザーには、AltStore経由でのインストールが最も確実な方法です。以下の手順で進めてください：

1
AltServerをPCにインストール
Windows 10以降、またはmacOS
2
iPhoneをPCに接続し、AltStoreをインストール
3
AltStore内の「Browse」タブでDelta Emulatorを検索
4
「GET」をタップしてインストール完了

public 方法3: AltStore PAL（欧州ユーザー向け）

EUに居住している場合、AltStore PALを利用することでPCなしでDeltaをインストールできます。EUのデジタル市場法（DMA）により、サードパーティのアプリストアがiOSでも認められています。

Delta自体にはゲームROMは含まれていません。法的に所有しているゲームのROMファイルを追加する必要があります。対応フォーマットは各プラットフォームの標準的な形式（.nes, .sfc, .z64, .gb, .gbc, .gba, .nds, .mdなど）です。

1
ROMファイルをiPhone/iPadの「ファイル」アプリに保存
2
Deltaを起動し、右上の「+」ボタンをタップ
3
「ファイル」から該当するROMファイルを選択
4
自動的にライブラリに追加されます

Nintendo DSについて：以前はBIOSファイルが必須でしたが、現在のバージョンではBIOSファイルはオプションとなっています。また、GBAカートリッジを「挿入」するデュアルスロット機能にも対応しています。

computer Part 2: UTM SE

UTM SEは、QEMUベースのフル機能システムエミュレータおよび仮想マシンホストです。iPhoneやiPadでWindows、Linux、その他のOSを実行することができます。「SE」は「Slow Edition」の略で、App Storeの制限に対応するためJITコンパイルの代わりにTCGインタープリターを使用しています。

2024年に登場したUTM SEは、iOS App Store初のPCエミュレータとなりました。通常のUTMはサイドローディングで入手可能ですが、AppleのJIT制限によりApp Storeでは配布できませんでした。UTM SEはこの制限を回避するため、パフォーマンスは低下しますが、脱獄やJIT回避策なしで使用可能です。

UTM SEはQEMUが対応する30以上のプロセッサアーキテクチャをエミュレート可能です：

x86_64 / i386

Windows、Linuxの多くのディストリビューション

ARM64 / ARM

ARMベースのOS、Raspberry Pi OS

RISC-V

RISC-V64 Ubuntuなど

PowerPC

クラシックMac OS 8.x/9.x

その他

MIPS、SPARC、m68kなど

UTM SEはiOS 14.0以降を搭載したデバイスで動作します。App Storeから「UTM SE」で検索してダウンロードするか、AltStore経由でインストールできます。

UTM SEでは様々なバージョンのWindowsを実行できますが、パフォーマンスの制約があるため、軽量なOSが推奨されます。

check_circle Windows 98/XPの場合（推奨）

Windows 98はUTM SEで驚くほどスムーズに動作します。M1 iPad Proでは、Pentium 120相当のパフォーマンスが得られます。90年代のWindowsソフトウェアやDOSゲームを楽しむには最適な選択です。

2-4GB

技術的には動作しますが、JITなしのエミュレーションではパフォーマンスが大幅に低下します。基本的な操作は可能ですが、実用的とは言いがたい速度です。

現時点では日本のApp Storeでは公開されていません。AltStore経由でのインストールが日本ユーザーにとって最も確実な方法です。米国アカウントをお持ちの場合は、地域を変更してダウンロードすることも可能です。

App Store版のUTM SEはJITコンパイルの代わりにTCGインタープリターを使用するため、パフォーマンスが低下します。AltStore経由でインストールしたUTMはJITが使えるため高速に動作します。実用的な速度を求める場合は、AltStore経由のUTMがおすすめです。

自分が所有しているカートリッジからリッピングしたROMファイルは個人使用の範囲で問題ないとされますが、インターネット上からの無断ダウンロードは著作権侵害になる可能性があります。詳しくはエミュレータの法的側面に関する記事をご参照ください。

はい、iOS 12以降を搭載したすべてのiPhoneで動作します。iPhone 15 / 16シリーズでも問題なく使用できます。新しい端末ほど高いパフォーマンスが期待できます。
