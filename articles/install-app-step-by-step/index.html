install_desktop アプリのインストール方法

AltStoreを使ってiPhoneにアプリをインストールする方法を、初心者向けにステップバイステップで解説。AltServerの準備から実際のアプリインストール、トラブル対応まで。

lightbulb AltStoreでアプリを入れるって難しい？

「AltStoreの話をよく聞くけど、実際にアプリを入れるには何をすればいいの？」「PC/Macって絶対に必要なの？」「インストール後のメンテナンスはどうするの？」

実は、AltStore でアプリをインストールするのは、App Store よりもシンプルで直感的です。 この記事では、初心者の方でも迷わないようにステップバイステップで解説します。 読み終わる頃には、最初のアプリをインストールし、定期メンテナンスの方法も理解できているはずです。

checklist 事前準備：インストール前に確認すること

AltStore はiOS 14.4 以上で動作します。

  • 設定 → 一般 → ソフトウェア・アップデートでバージョンを確認
  • iOS 14.4 未満の場合はアップデートが必要

AltStore をインストールするには、Windows PC または Macが必要です。 初回セットアップ時のみ USB 接続が必要で、その後は Wi-Fi 経由での操作が可能になります。

  • Windows: Windows 10 以上推奨
  • Mac: macOS 10.14 以上推奨

有効な Apple ID（2段階認証を有効にしたもの）が必要です。 既に iPhone で使用している Apple ID で問題ありません。

info 重要：Developer Accountは不要

有料の Developer Account は必要ありません。無料の Apple ID で大丈夫です。

cable 初回のみUSB接続が必要

初回セットアップ時のみ、iPhone をPC/Mac に USB で接続します。 その後は Wi-Fi 経由での操作になるため、毎回 USB 接続は不要です。

construction セットアップ5ステップ

1

AltServerをダウンロード＆インストール

公式サイトからPC/Mac用のAltServerをダウンロードしてインストール。特別な設定は不要です。

info AltServerのダウンロード先
AltServer公式サイトの手順を参照してください。Macの手順・Windowsの手順が公式FAQに掲載されています（altstore.io）。
2

AltServerにApple IDでログイン

AltServerのメニューから「Sign In」を選択し、iPhoneで使用しているApple IDでサインイン。2段階認証コードの入力が必要な場合があります。

3

iPhoneをUSBで接続してAltStoreをインストール

iPhoneをUSBケーブルでPCに接続 → AltServerメニューから「Install AltStore」を選択 → 数秒でインストール完了。

info Step 3：「信頼」をタップすることを忘れずに
USB接続時に「このコンピューターを信頼しますか？」というメッセージが iPhone に表示される場合は、必ず「信頼」をタップしてください。これをしないとインストールが進みません。
4

iPhoneのAltStoreを信頼する

設定 → 一般 → VPNとデバイス管理で自身のApple IDの欄をタップ。「信頼」をタップして完了。

5

Wi-Fi経由での接続を有効化

Finderで「Wi-Fi経由でiPhoneとの接続」を有効にすれば、以降はUSB接続が不要になります。

cable 接続トラブルの9割はケーブルが原因
100均の「充電専用」ケーブルでは AltServer は認識しません。必ずデータ転送対応のケーブルを使ってください。
おすすめの高耐久ケーブルを見る open_in_new

apps アプリのインストール方法

1

iPhone 上の AltStore アプリを開く

2

「Browse」タブでアプリ一覧を表示

Tweaked Apps、Emulators など

3

インストールしたいアプリをタップ

4

詳細ページの「Get」ボタンをタップ

5

ダウンロード完了後、「Open」ボタンが表示されればインストール成功

AltStore 内のアプリ一覧以外にも、IPA ファイル（iPhone アプリのパッケージファイル） を PC/Mac で用意して、AltServer からインストールする方法もあります。 カスタムアプリや個人開発アプリをインストールしたい場合に便利です。

1

信頼できるソースからIPAファイルを入手

2

AltServer メニューを開き「Sideload .ipa」などを選択

3

IPAファイルを指定してインストール

wifi Wi-Fi 経由での自動更新設定（推奨）

autorenew 自動更新で手間いらず

AltStore のアプリは7日ごとに署名を更新する必要があります。 毎回 USB で接続する代わりに、Wi-Fi 経由で自動更新するように設定すると、 手動操作がほぼ不要になります。

1

PC/Mac の AltServer メニューを開く

2

インストール済みの iPhone を選択

3

「Sync with [iPhone name] over Wi-Fi」オプションをオンにする

  • PC/Mac と iPhone が同じ Wi-Fi ネットワークに接続していること
  • AltServer がバックグラウンド起動していること
  • iPhone が同じ Wi-Fi に常に接続していることが理想的

Wi-Fi 同期を有効にすると、AltServer は以下の処理を自動的に行います：

1

毎日、iPhone 上のアプリの署名有効期限をチェック

2

期限切れ間近のアプリを検出

3

Wi-Fi 経由で自動的に再署名（更新）

4

ユーザーは何もしなくてOK

つまり、PC を起動して AltServer を実行しておくだけで、アプリは常に最新状態に保たれるわけです。

build トラブル対応：よくある問題と解決策

  • Apple ID のパスワードが正しく入力されているか
  • iOS が 14.4 以上か
  • iPhone の容量は十分か（アプリサイズ以上の空きが必要）
  • AltServer を再起動してみる

署名が切れている可能性があります。

解決法：AltStore アプリを開き、該当アプリの横に表示されている更新ボタン（↻）をタップして、 署名を更新してください。通常、数分で起動可能になります。

  • PC/Mac と iPhone が同じ Wi-Fi ネットワークに接続しているか
  • AltServer が PC 上でバックグラウンド実行中か（メニューバーに表示されているか）
  • Wi-Fi ルーターのファイアウォール設定が AltServer の通信をブロックしていないか
  • PC を再起動して、再度 Wi-Fi 同期をオンにしてみる

家族や友人の PC/Mac を借りて初回セットアップを完了させれば、その後は iPhone 上で手動で署名を更新できます。毎回 USB 接続は不要です。 また、SideStoreに切り替えることでPC不要での運用も可能です。

AltServer と AltStore に同じ Apple ID でサインインしてください。複数の Apple ID は対応していません。

update 定期メンテナンス：署名更新の方法

schedule 重要：7日ごとの署名更新を忘れずに

AltStore のアプリは7日ごとに署名が切れます。 Wi-Fi 同期が有効なら、期限切れの2～3日前に自動更新されます。 手動更新の場合は、7日以内なら随時更新できます。期限を過ぎるとアプリが起動しなくなりますので注意してください。

touch_app
手動更新（Wi-Fi同期なし）
AltStoreアプリを開き、アプリの横の更新ボタン（↻）をタップ。数分で完了。
autorenew
Wi-Fi経由の自動更新（推奨）
ステップ5の設定完了後は何もしなくても自動更新されます。PCを起動してAltServerを実行しておくだけでOK。
