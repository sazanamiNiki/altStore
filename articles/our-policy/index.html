policy 当サイトについて・運営方針

bunchoniki Storeの運営方針と掲載基準。オープンソースまたはApple公証により安全性を確認したアプリのみを厳選し、初心者でも安全に利用できる情報を提供します。

lightbulb なぜこのサイトを作ったのか

iPhoneのサイドローディングは、App Storeでは提供されない便利なアプリを使える画期的な仕組みです。しかし、日本語の情報が少なく、初心者には敷居が高いという課題がありました。

また、ネット上には出どころ不明なアプリや安全性が不明なアプリの情報も混在しており、どの情報を信じればいいのか分からないという声も多く聞かれます。

bunchoniki Storeは、「開発元が明確なアプリのみを扱う」「初心者でも安心して使える」「日本語で分かりやすく解説する」という3つの方針のもと、サイドローディングの正しい知識と安全な使い方を広めることを目的として運営しています。

stars 当サイトの特徴

当サイトでは、オープンソースまたはApple公証により安全性を確認したアプリのみを紹介しています。この2つの方法により、開発元が明確で、マルウェアやウイルスが含まれていないことを確認しています。

code オープンソースによる透明性

エミュレータなどのオープンソースアプリは、ソースコードが公開されており、誰でも内部の動作を確認できます。コードが公開されているため悪意あるコードが混入しにくく、世界中の開発者がコードをレビューし、GitHubなどで開発履歴も確認できます。

shield Apple公証による信頼性

クリエイター向けツールなど、商用アプリの多くはAppleの公証（Notarization）を受けています。公証済みアプリは、Appleが安全性をチェックし、マルウェアやウイルスが含まれていないことを確認したアプリです。開発元の身元がApple Developer IDで確認されており、問題が発見された場合は証明書を取り消すことができます。

warning 出どころ不明のアプリのリスク

マルウェア・ウイルスの混入、個人情報の不正な収集・送信、デバイスの動作不良、法的・規約的な問題が発生するリスクがあります。当サイトではオープンソースまたは公証済みのアプリのみを扱い、ユーザーの安全を最優先に考えています。

サイドローディングは、専門用語や技術的な知識が必要に思えるかもしれません。しかし、当サイトでは難しい言葉を避け、手順を詳しく解説することで、初心者でも安心して利用できるようにしています。

また、各記事には「よくある質問」や「トラブルシューティング」を用意し、つまずきやすいポイントをカバーしています。

サイドローディングの情報は英語が中心で、日本語の情報源が限られています。当サイトでは、日本語で分かりやすく、かつ正確な情報を提供することを心がけています。

技術的な専門用語も、必要に応じて日本語の説明を付けています。

rule 掲載基準

すべての掲載アプリは、オープンソースまたはApple公証により安全性を確認している必要があります。具体的には、以下の方法で確認しています：

code
オープンソースアプリの確認
  • GitHubなどでソースコードが公開されている
  • 開発履歴やコミット内容が確認できる
  • コミュニティによる活発な開発・レビューが行われている
  • 実際にコードをレビューし、問題がないことを確認
verified
公証済みアプリの確認
  • 公式サイトやGitHubリポジトリでの公式アナウンス
  • 実際にインストールして動作確認
  • Apple Developer IDの確認

掲載アプリは、実際に役立つ機能を持っていることを重視しています。単なる実験的なアプリや、使い道が限定的なものは対象外です。

開発が停止しているアプリや、長期間更新されていないアプリは、セキュリティリスクが高まります。当サイトでは、継続的に開発・保守されているアプリを優先的に掲載しています。

fact_check 情報の信頼性

当サイトの情報は、以下のソースをもとに作成しています：

  • Apple公式ドキュメント：サイドローディングやApp Storeの規約に関する公式情報
  • 各アプリの公式サイト・GitHubリポジトリ：最新のバージョン情報や機能説明、ソースコードの確認
  • 実際の検証：運営者が実際にインストール・テストして確認
  • 法的情報：デジタル市場法（DMA）や日本の法律に関する信頼できる情報源

サイドローディングの情報は急速に変化します。当サイトでは、最新情報を定期的に更新し、古い情報を放置しないように努めています。

各記事の最後には「最終更新日」を明記し、情報の新しさを確認できるようにしています。

warning サイドローディングは自己責任で行ってください

当サイトで紹介する情報は、正確性と安全性を重視していますが、すべての動作を保証するものではありません。

bunchoniki Storeは、以下の点について責任を負いかねます：

  • デバイスの故障や不具合：サイドローディングの実行により生じたデバイスの問題
  • データの損失：アプリのインストールや使用によるデータ消失
  • 法的問題：利用者の居住地域の法律に基づく問題（利用者自身で確認してください）
  • アプリの動作：紹介したアプリが期待通りに動作しない場合
  • Appleによるペナルティ：Apple IDの停止や利用制限（通常は発生しませんが、規約違反があった場合）

サイドローディングを利用する際は、以下の点をご理解ください：

  • 当サイトの情報を参考にする場合でも、最終的な判断と責任は利用者自身にあります
  • アプリのインストール前に、公式サイトや利用規約を必ず確認してください
  • 法的に問題のないアプリのみを使用してください（違法コピーアプリやチートツールは対象外です）
  • 重要なデータは定期的にバックアップを取ってください

当サイトは最新かつ正確な情報の提供に努めていますが、情報が古くなる可能性や、誤りが含まれる可能性があります。重要な判断をする際は、必ず公式情報を確認してください。
