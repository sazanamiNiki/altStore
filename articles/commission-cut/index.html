payments 手数料カットの真実

Apple が手数料を 30% から 20% に削減。でもアプリは安くなるのか？EU の実データから、その現実と AltStore の本当のメリットを解説します。

lightbulb 「手数料が下がればアプリも安くなるはず」と期待していませんか？

Apple が App Store の手数料を 30% から 20% に削減 した。ニュースを聞いて、こう思いませんでしたか？

「手数料が下がるなら、その分ユーザーにとってアプリが安くなるはず」

残念ながら、その期待はほぼ叶いません。 実際の EU のデータでは、手数料削減後も 90% 以上のアプリは価格が変わりませんでした。 この記事では、その現実と、AltStore や SideStore が提供する本当のメリットについて解説します。

trending_down 衝撃の事実：EU での手数料削減後も、アプリは安くならなかった

欧州連合（EU）は DMA（Digital Markets Act）により、Apple に対して手数料の削減を強制しました。 2024 年 3 月から徐々に、2025 年 6 月には本格的に 20% に削減されたのです。

ところが、実際に起きたことは予想と真逆でした。 手数料削減から数ヶ月が経った今、以下のデータが示されています：

bar_chart EU アプリストアの価格変動データ
  • 90% 以上：手数料削減後も価格は変わらず
  • 一部の例外：Spotify などの大手がわずかに価格調整
  • 開発者の多くは：削減分を利益として確保

30% の手数料の中には、実際の運営コストが含まれていました。 その削減分は単なる「余利」ではなく、開発チームの給料や新機能開発に充てられるのです。

  • エンジニアの給与：手数料削減分で初めてプラマイゼロ
  • サーバー・インフラ：ユーザー数の増加に対応する必要がある
  • セキュリティ対策：脆弱性パッチや安全性向上に継続投資が必須
  • 新機能開発：競合アプリに対抗するための R&D 費用

実は、アプリの価格設定には暗黙のルールがあります。

  • 「9.99 ドル」の壁：$9.99 と $8.99 では心理的な印象が全く異なる
  • 「499 円」の壁：500 円と 499 円でも購入意欲が大きく変わる
  • 心理的割引：「299 円」の方が「300 円」より安く感じる

開発者は慎重に価格を決めているため、手数料削減といえど、 安易に価格を 10% 下げるような価格改定は避けられるのです。

多くの開発者にとって、30% の手数料は「家賃」のようなもの。 つまり、その手数料を差し引いた後の売上でも、利益は最小限だったのです。

手数料が 30% → 20% に下がったことで、ようやく：

  • 赤字だったプロジェクトがプラマイゼロになる
  • 細い利益がやや太くなる
  • 新機能開発に投資する余裕が出る

要するに、ユーザーへの値下げではなく、「生き残りのための手段」として使われているのです。

verified 例外的に価格が安くなるケース：Web 課金の場合

「アプリは安くならない」と言いながらも、例外が存在します。Spotify などの大手サービスです。 これらのサービスが可能にした理由：

  • Web サイトからの課金：App Store を迂回した支払い方法を提供
  • Core Technology Fee（5%）：App Store の手数料より大幅に安い
  • 大規模なユーザーベース：数百万人のユーザーがいるため、少額の値下げでも効果が大きい

DMA 対応により、開発者はApp Store と Web 課金の両方を提供できるようになりました。

  • Web サイト経由なら 5%（CTF のみ）で課金できる
  • App Store 経由なら 20%が手数料として取られる
  • ユーザーが Web サイトを使うようになると、開発者の収益が増える

したがって、長期的には一部の大手サービス（特に継続課金型）は、 Web 課金での価格競争が起きる可能性があります。 ただし、通常のアプリはほぼ変化なしというのが現状です。

diamond AltStore・SideStore がもたらす本当のメリット：手数料ゼロの直接支援

答えは単純です。AltStore は「手数料がない」。 つまり、開発者とユーザーが直接つながるプラットフォームなのです。

Apple の手数料削減は、単なる「値下げ」ではなく、 デジタルマーケットプレイスの構造そのものが変わろうとしていることを示しています。
