<Section title="例外的に価格が安くなるケース：Web 課金の場合" icon="verified">
  <AccordionGroup>
    <Accordion title="Spotify はなぜ安くなったのか？">
      <p class="text-gray-700 mb-4">
        「アプリは安くならない」と言いながらも、<strong>例外が存在します</strong>。Spotify などの大手サービスです。
        これらのサービスが可能にした理由：
      </p>

        <li><strong>Web サイトからの課金：</strong>App Store を迂回した支払い方法を提供</li>
        <li><strong>Core Technology Fee（5%）：</strong>App Store の手数料より大幅に安い</li>
        <li><strong>大規模なユーザーベース：</strong>数百万人のユーザーがいるため、少額の値下げでも効果が大きい</li>
      </ul>
    </Accordion>
    <Accordion title="Web 課金がもたらす価格競争">
      <p class="text-gray-700 mb-4">
        DMA 対応により、開発者は<mark class="bg-yellow-100 px-1 rounded"><strong>App Store と Web 課金の両方を提供</strong></mark>できるようになりました。
      </p>
      <ul class="list-disc pl-6 space-y-2 text-gray-700 mb-4">
        <li><strong>Web サイト経由なら 5%（CTF のみ）</strong>で課金できる</li>
        <li><strong>App Store 経由なら 20%</strong>が手数料として取られる</li>
        <li>ユーザーが Web サイトを使うようになると、開発者の収益が増える</li>
      </ul>
      <p class="text-gray-700">
        したがって、<strong>長期的には</strong>一部の大手サービス（特に継続課金型）は、
        Web 課金での価格競争が起きる可能性があります。
        ただし、通常のアプリはほぼ変化なしというのが現状です。
      </p>
    </Accordion>
  </AccordionGroup>
