<Section title="トラブルシューティング" icon="build" iconColor="text-red-600">
  <p class="text-gray-700 leading-relaxed sm:leading-7 mb-3 sm:mb-4">
    問題が発生したら、まず<strong>VPN接続と証明書の信頼設定</strong>を確認してください。
  </p>
  <CardGrid>
    <Card icon="vpn_key" iconColor="text-purple-500" title="VPN確認" desc="接続状態をチェック" />
    <Card icon="verified_user" iconColor="text-green-500" title="証明書確認" desc="設定→VPNとデバイス管理" />
  </CardGrid>
  <AccordionGroup>
    <Accordion title="「Unable to Install」エラー">
      <p class="text-gray-700 mb-2"><strong>原因：</strong>VPN未接続または証明書未信頼</p>
      <ul class="list-disc pl-4 sm:pl-6 space-y-1.5 sm:space-y-2 text-gray-700">
        <li>VPNアプリで接続状態を確認</li>
        <li>「設定」→「一般」→「VPNとデバイス管理」で証明書を信頼</li>
        <li>SideStoreとVPNアプリを再起動</li>
      </ul>
    </Accordion>
    <Accordion title="アプリが7日後に開けなくなった">
      <p class="text-gray-700 mb-2"><strong>原因：</strong>証明書の有効期限切れ</p>
      <ul class="list-disc pl-4 sm:pl-6 space-y-1.5 sm:space-y-2 text-gray-700">
        <li>VPNを接続してSideStoreを起動</li>
        <li>「My Apps」でアプリをリフレッシュ</li>
        <li>SideStore自体が開けない場合はiLoaderで再インストール</li>
      </ul>
    </Accordion>
    <Accordion title="VPNが接続できない">
      <p class="text-gray-700 mb-2"><strong>原因：</strong>VPN構成またはネットワークの問題</p>
      <ul class="list-disc pl-4 sm:pl-6 space-y-1.5 sm:space-y-2 text-gray-700">
        <li>「設定」→「VPNとデバイス管理」→「VPN」でプロファイルを削除し再設定</li>
        <li>LocalDevVPNの代わりにStosVPNを試す</li>
        <li>デバイスを再起動</li>
      </ul>
    </Accordion>
    <Accordion title="「Maximum App ID Limit Reached」エラー">
      <p class="text-gray-700 mb-2"><strong>原因：</strong>週間アプリ制限（10個）に達した</p>
      <ul class="list-disc pl-4 sm:pl-6 space-y-1.5 sm:space-y-2 text-gray-700">
        <li>7日間待ってから再試行</li>
        <li>不要なアプリを削除してスロットを空ける</li>
        <li>LiveContainerの使用を検討</li>
      </ul>
    </Accordion>
    <Accordion title="Developer Modeが表示されない">
      <p class="text-gray-700 mb-2"><strong>原因：</strong>iOS 15以前、または未サイドロード</p>
      <ul class="list-disc pl-4 sm:pl-6 space-y-1.5 sm:space-y-2 text-gray-700">
        <li>iOS 15以前では Developer Mode は不要</li>
        <li>iOS 16以降では、iLoaderでインストールを試みると表示される</li>
        <li>Xcodeインストール済みMacに接続すると表示される場合あり</li>
      </ul>
    </Accordion>
  </AccordionGroup>
  <InfoBox type="info" icon="tips_and_updates" title="その他のヒント">
