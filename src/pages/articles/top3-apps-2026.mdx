<Section title="2位：UTM SE" icon="computer" iconColor="text-gray-600">
  <p class="text-gray-500 mb-4">開発者: UTM Project | カテゴリ: 仮想化・エミュレーション</p>
  <AccordionGroup>
    <Accordion title="概要">
      <p class="text-gray-700 mb-4">
        UTM SEは、QEMUをベースにした仮想マシンアプリです。「SE」は「Slow Edition」の略で、App Storeの制限（JITコンパイル禁止）に対応するため、TCGインタープリターを使用しています。2024年にiOS App Store初のPCエミュレータとして登場し、iPhoneやiPadでWindows、Linux、その他のOSを実行することが可能になりました。
      </p>
      <p class="text-gray-700">
        x86_64、ARM64、RISC-V、PowerPCなど30以上のCPUアーキテクチャをエミュレートでき、Windows 98からWindows 11、各種Linuxディストリビューション、クラシックMac OS 8.x/9.xまで幅広いOSに対応しています。パフォーマンスはネイティブには及びませんが、軽量なOSやレガシーソフトウェアの実行には十分実用的です。
      </p>
    </Accordion>
    <Accordion title="評価ポイント：優れている点と注意点">
      <CardGrid>
        <Card icon="check_circle" iconColor="text-green-500" title="優れている点">
          <ul class="list-disc pl-4 space-y-1 text-gray-600 text-xs mt-1">
            <li>30以上のCPUアーキテクチャ対応</li>
            <li>Windows/Linux/Mac OSなど多数のOS対応</li>
            <li>App Storeから正式入手可能</li>
            <li>脱獄・JIT回避策不要</li>
            <li>活発な開発とアップデート</li>
            <li>改善されたセットアップウィザード</li>
            <li>Shortcuts自動化対応</li>
            <li>iOS 26 Liquid Glassデザイン対応</li>
          </ul>
        </Card>
        <Card icon="warning" iconColor="text-orange-500" title="注意点">
          <ul class="list-disc pl-4 space-y-1 text-gray-600 text-xs mt-1">
            <li>パフォーマンスは低め（Pentium 120相当）</li>
            <li>GPU仮想化非対応（DirectX/OpenGL不可）</li>
            <li>現代のWindows 10/11は非実用的</li>
            <li>セットアップにはある程度の知識が必要</li>
            <li>ストレージ容量を多く消費</li>
          </ul>
        </Card>
      </CardGrid>
    </Accordion>
    <Accordion title="具体的な使用例">
      <ul class="space-y-4">
        <li class="border-l-4 border-gray-400 pl-4">
          <strong class="block mb-1">レトロPCゲームのプレイ</strong>
          <p class="text-gray-700 text-sm">Windows 98をインストールし、90年代のPCゲームやDOSゲームをiPadでプレイ。M1 iPad ProではPentium 120相当のパフォーマンスが得られ、シムシティ2000やAge of Empires IIなど、懐かしのタイトルを楽しめる。</p>
        </li>
        <li class="border-l-4 border-gray-400 pl-4">
          <strong class="block mb-1">Linux学習環境の構築</strong>
          <p class="text-gray-700 text-sm">Alpine LinuxやDebian最小インストールを使用し、コマンドライン操作やサーバー管理の学習環境を構築。出先でもターミナル操作の練習が可能。SSH接続用のJumpサーバーとしても活用できる。</p>
        </li>
        <li class="border-l-4 border-gray-400 pl-4">
          <strong class="block mb-1">レガシーソフトウェアの実行</strong>
          <p class="text-gray-700 text-sm">古いWindowsでしか動作しない業務ソフトや、特定のハードウェア構成が必要なツールを実行。クラシックMac OS 9を使用してHyperCardスタックを開いたり、古いFileMakerデータベースにアクセスしたりすることも可能。</p>
        </li>
      </ul>
    </Accordion>
  </AccordionGroup>
  <InfoBox type="info" icon="computer" title="総合評価">

  </InfoBox>
</Section>

<Section title="3位：SideStore" icon="phone_iphone" iconColor="text-orange-600">
  <p class="text-gray-500 mb-4">開発者: SideStore Team | カテゴリ: アプリ管理・サイドローディング</p>
  <AccordionGroup>
    <Accordion title="概要">
      <p class="text-gray-700 mb-4">
        SideStoreは、AltStoreのフォーク（派生版）として開発されたコミュニティ主導のサイドローディングアプリです。最大の特徴は、初回セットアップ後はPCなしでアプリのインストールや更新が可能な点。VPN技術（EM Proxy）を活用することで、iOSの制限を回避し、完全にデバイス上でサイドローディングを完結させることができます。
      </p>
      <p class="text-gray-700">
        iOS 14からiOS 18.4まで対応しており、LocalDevVPNまたはStosVPNと組み合わせて使用します。AltStoreの利便性を継承しながら、PC依存を最小限に抑えた革新的なソリューションです。2026年1月時点の最新版は v0.6.3 で、AltStore 2.0の変更点も取り込まれています。
      </p>
    </Accordion>
    <Accordion title="評価ポイント：優れている点と注意点">
      <CardGrid>
        <Card icon="check_circle" iconColor="text-green-500" title="優れている点">
          <ul class="list-disc pl-4 space-y-1 text-gray-600 text-xs mt-1">
            <li>初回セットアップ後はPC不要</li>
            <li>VPNはローカル接続のみ（外部通信なし）</li>
            <li>バッテリー消費がほぼゼロ</li>
            <li>カスタムソースの追加が可能</li>
            <li>IPAファイルから直接インストール</li>
            <li>バックグラウンド自動リフレッシュ</li>
            <li>活発なコミュニティ開発</li>
            <li>AltStore 2.0機能を継承</li>
          </ul>
        </Card>
        <Card icon="warning" iconColor="text-orange-500" title="注意点">
          <ul class="list-disc pl-4 space-y-1 text-gray-600 text-xs mt-1">
            <li>初期セットアップはやや複雑</li>
            <li>VPN設定が必要（LocalDevVPN/StosVPN）</li>
            <li>無料Apple IDは3アプリ制限あり</li>
            <li>7日ごとのリフレッシュが必要</li>
            <li>AltStoreより技術的知識が必要</li>
          </ul>
        </Card>
      </CardGrid>
    </Accordion>
    <Accordion title="具体的な使用例">
      <ul class="space-y-4">
        <li class="border-l-4 border-orange-400 pl-4">
          <strong class="block mb-1">PCなしでの完全モバイル運用</strong>
          <p class="text-gray-700 text-sm">自宅にPCがない、または普段使わないユーザーに最適。初回セットアップさえ済ませば、その後はiPhoneだけでアプリの追加・更新・管理が完結。旅行中やPC故障時も問題なくアプリを維持できる。</p>
        </li>
        <li class="border-l-4 border-orange-400 pl-4">
          <strong class="block mb-1">開発者・テスターのベータアプリ管理</strong>
          <p class="text-gray-700 text-sm">開発中のアプリやベータ版をIPAファイルから直接インストール。TestFlightを使わずにアプリ配布が可能。カスタムソースを追加すれば、チーム内でのアプリ共有も簡単に。</p>
        </li>
        <li class="border-l-4 border-orange-400 pl-4">
          <strong class="block mb-1">AltStoreアプリの補助ツールとして</strong>
          <p class="text-gray-700 text-sm">AltStoreをメインで使いながら、PC接続が難しい期間のバックアップとしてSideStoreを活用。両方をインストールしておくことで、状況に応じて使い分けが可能。緊急時のリフレッシュ手段として保険的に利用。</p>
        </li>
      </ul>
    </Accordion>
  </AccordionGroup>
  <InfoBox type="success" icon="phone_iphone" title="総合評価">
    AltStoreの「PC接続が面倒」という唯一の欠点を解消したアプリ。セットアップはやや複雑だが、一度完了すれば自由度の高いアプリ管理が実現。PCへのアクセスが限られている方には必須のツール。
  </InfoBox>
</Section>
