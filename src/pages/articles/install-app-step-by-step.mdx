      <ul class="list-disc pl-6 space-y-2 text-gray-700">

      </ul>
    </Accordion>
    <Accordion title="PC / Mac の用意">
      <p class="text-gray-700 mb-3">
        AltStore をインストールするには、<strong>Windows PC または Mac</strong>が必要です。
        初回セットアップ時のみ USB 接続が必要で、その後は Wi-Fi 経由での操作が可能になります。
      </p>
      <ul class="list-disc pl-6 space-y-2 text-gray-700">

      </p>
      <InfoBox type="info" title="重要：Developer Accountは不要">

    <Accordion title="USB ケーブル（初回セットアップのみ）">
      <InfoBox type="success" icon="cable" title="初回のみUSB接続が必要">
        初回セットアップ時のみ、iPhone をPC/Mac に USB で接続します。
        その後は Wi-Fi 経由での操作になるため、毎回 USB 接続は不要です。
      </InfoBox>
    </Accordion>

<Section title="Wi-Fi 経由での自動更新設定（推奨）" icon="wifi">
  <InfoBox type="success" icon="autorenew" title="自動更新で手間いらず">
    AltStore のアプリは<strong>7日ごとに署名を更新</strong>する必要があります。
    毎回 USB で接続する代わりに、<strong>Wi-Fi 経由で自動更新</strong>するように設定すると、
    手動操作がほぼ不要になります。
  </InfoBox>
  <AccordionGroup>
    <Accordion title="AltServer側での設定方法">
      <StepList>
        <Step num={1} title="PC/Mac の AltServer メニューを開く" />
        <Step num={2} title="インストール済みの iPhone を選択" />
        <Step num={3} title="「Sync with [iPhone name] over Wi-Fi」オプションをオンにする" />
      </StepList>
    </Accordion>
    <Accordion title="Wi-Fi同期を機能させるための必須条件">
      <ul class="list-disc pl-6 space-y-2 text-gray-700">
        <li><strong>PC/Mac と iPhone が同じ Wi-Fi ネットワークに接続</strong>していること</li>
        <li><strong>AltServer がバックグラウンド起動</strong>していること</li>
        <li><strong>iPhone が同じ Wi-Fi に常に接続</strong>していることが理想的</li>
      </ul>
    </Accordion>
    <Accordion title="自動更新の仕組み">
      <p class="text-gray-700 mb-3">Wi-Fi 同期を有効にすると、AltServer は以下の処理を自動的に行います：</p>
      <StepList>
        <Step num={1} title="毎日、iPhone 上のアプリの署名有効期限をチェック" />
        <Step num={2} title="期限切れ間近のアプリを検出" />
        <Step num={3} title="Wi-Fi 経由で自動的に再署名（更新）" />
        <Step num={4} title="ユーザーは何もしなくてOK" />
      </StepList>
      <p class="text-gray-700 mt-3">
        つまり、<strong>PC を起動して AltServer を実行しておくだけで、アプリは常に最新状態に保たれる</strong>わけです。
      </p>
    </Accordion>
  </AccordionGroup>
</Section>

<Section title="トラブル対応：よくある問題と解決策" icon="build" iconColor="text-red-600">
  <AccordionGroup>
    <Accordion title="インストールが失敗する">
      <ul class="list-disc pl-6 space-y-2 text-gray-700">
        <li>Apple ID のパスワードが正しく入力されているか</li>
        <li>iOS が 14.4 以上か</li>
        <li>iPhone の容量は十分か（アプリサイズ以上の空きが必要）</li>
        <li>AltServer を再起動してみる</li>
      </ul>
    </Accordion>
    <Accordion title="インストール後、アプリが起動しない">
      <p class="text-gray-700 mb-2">署名が切れている可能性があります。</p>
      <p class="text-gray-700">
        <strong>解決法：</strong>AltStore アプリを開き、該当アプリの横に表示されている更新ボタン（↻）をタップして、
        署名を更新してください。通常、数分で起動可能になります。
      </p>
    </Accordion>
    <Accordion title="Wi-Fi 自動更新がうまくいかない">
      <ul class="list-disc pl-6 space-y-2 text-gray-700">
        <li>PC/Mac と iPhone が<strong>同じ Wi-Fi ネットワーク</strong>に接続しているか</li>
        <li>AltServer が PC 上でバックグラウンド実行中か（メニューバーに表示されているか）</li>
        <li>Wi-Fi ルーターのファイアウォール設定が AltServer の通信をブロックしていないか</li>
        <li>PC を再起動して、再度 Wi-Fi 同期をオンにしてみる</li>
      </ul>
    </Accordion>
    <Accordion title="PC がない場合はどうするの？">
      <p class="text-gray-700">
        家族や友人の PC/Mac を借りて初回セットアップを完了させれば、その後は iPhone 上で手動で署名を更新できます。毎回 USB 接続は不要です。
        また、SideStoreに切り替えることでPC不要での運用も可能です。
      </p>
    </Accordion>
    <Accordion title="Apple ID が複数ある場合は？">
      <p class="text-gray-700">
        AltServer と AltStore に同じ Apple ID でサインインしてください。複数の Apple ID は対応していません。
      </p>
    </Accordion>
  </AccordionGroup>
</Section>

<Section title="定期メンテナンス：署名更新の方法" icon="update">
  <InfoBox type="error" icon="schedule" title="重要：7日ごとの署名更新を忘れずに">
    AltStore のアプリは<strong>7日ごと</strong>に署名が切れます。
    Wi-Fi 同期が有効なら、期限切れの2～3日前に自動更新されます。
    手動更新の場合は、7日以内なら随時更新できます。期限を過ぎるとアプリが起動しなくなりますので注意してください。
  </InfoBox>
